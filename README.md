# Droppr
A little helper for custom select creation.

# Install
```sh
bower install droppr
```

# Usage
```js
window.Droppr({
  'id': '#select', //parent element of your custom select
  'itens': 'ul li', //your option list 
  'onChange': function() {
    // your handler here
  }
});
```
